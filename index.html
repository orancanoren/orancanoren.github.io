CRYPTORAN

A pure Python 3 cryptographic algorithms library.
Available as a Python 3 package and CLI tool.

  • Block ciphers: AES-128, DES
  • Public-key cryptosystems: El Gamal, RSA
  • Key exchange: Diffie-Hellman
  • Signatures: RSA signature

RSA-OAEP is supported.
Block ciphers support CBC and ECB modes of operation.

LEARNORAN

A CPU & GPU parallelized machine learning API for C++.

Currently multiple regression is the only available model.
Training is done through stochastic gradient descent.

Parallelization of training is done on CPU and Nvidia GPU's using OpenMP and CUDA frameworks.
