CRYPTOGRAPHIC ALGORITHMS SUITE

Pure Python3 implementations of various cryptographic algorithms. No external dependencies!

  • Available block ciphers: DES
  • Available public-key cryptosystems: El Gamal, RSA
  • Available secret-key sharing schemes: Diffie-Hellman protocol

AES is currently under development. OAEP encoding has also been developed and integrated with RSA.

PARALLEL HASHCASH
$$ H(k + n) = 0...00xxx...$$

Parallelized Hashcash (i.e. Bitcoin proof-of-work) computation tool. Parallelization is done on CPU and Nvidia GPU's using OpenMP and CUDA frameworks.

Uses SHA256 hashing. Thanks to Crypto++ community for providing high performance SHA256 implementation.
